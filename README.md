# Video Game Sales Data Analysis

## Description
Analyzed video game sales data using Python, Pandas, Matplotlib, and Seaborn. Calculated global sales statistics, filtered high-selling games, normalized sales data, and created visualizations including a histogram and NA vs EU sales comparison chart.

---

# Project Overview

This project performs Exploratory Data Analysis (EDA) on a video game sales dataset. The analysis focuses on understanding global and regional sales trends using Python data analysis and visualization libraries.
